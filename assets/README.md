# Dossier Assets

Placez ici vos fichiers d'images:

## Fichiers requis:

1. **logo.png** - Logo de Hexacity (24x24px recommandé)
   - Affiché dans la barre de titre

2. **icon.png** - Icône de l'application (256x256px recommandé)
   - Utilisée pour l'application

3. **icon.ico** - Icône Windows (pour l'exécutable compilé)
   - À créer à partir de icon.png

## Comment créer ces images:

### logo.png
- Taille: 24x24px ou 48x48px
- Format: PNG avec transparence
- Utilisez votre logo de serveur

### icon.png
- Taille: 256x256px ou 512x512px
- Format: PNG avec transparence
- Peut être le même que le logo, mais en plus grand

### icon.ico
- Convertissez icon.png en .ico en ligne:
  - https://convertio.co/fr/png-ico/
  - https://www.icoconverter.com/

## Images temporaires

Si vous n'avez pas encore vos images, le launcher fonctionnera quand même.
Les images manquantes n'empêchent pas le launcher de démarrer.
