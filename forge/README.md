# Dossier Forge/NeoForge

## Instructions:

1. Téléchargez NeoForge 21.1.206 pour Minecraft 1.21.1 depuis:
   https://neoforged.net/

2. Recherchez la version **21.1.206** pour Minecraft **1.21.1**

3. Téléchargez le fichier **Installer** (format .jar)

4. Placez le fichier téléchargé dans ce dossier

5. Renommez-le en: `neoforge-21.1.206-installer.jar`

## Alternative:

Si vous préférez utiliser le launcher Minecraft vanilla (sans mods), vous pouvez:

1. Supprimer la ligne `forge:` dans le fichier `main.js`
2. Le launcher lancera alors Minecraft 1.21.1 vanilla

## Note:

Le launcher peut fonctionner sans NeoForge en mode vanilla Minecraft.
NeoForge est nécessaire uniquement si votre serveur utilise des mods.
